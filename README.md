# spring_microservices
Kafka &amp; SOAP &amp; HTTP API

- prepare mysql database
- run zookeeper
- run kafka
- execute "mvn install" each service
- run each service with "mvn spring-boot:run"
- make HTTP queries to Gateway API with Postman or whatever
